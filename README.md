# UChile Schedule Planner

A web application designed to help you preview and plan your biannual schedule for each semester at the University of Chile.

## Purpose

The main goal of this project is to provide a clear and interactive way to visualize your semester courses, making it easier to manage your time and academic load.

## Features

- **Visual Schedule Grid**: Quickly see your weekly class distribution.
- **Course Management**: Add and organize your courses easily.
- **Semester Planning**: Tailored for biannual semester planning.

## Installation

To get this project running locally on your machine, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/iriake/uchile-schedule-planner.git
   cd uchile-schedule-planner
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

   The application will start, and you can view it in your browser.
